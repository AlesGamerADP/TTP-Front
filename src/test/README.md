# Tests

Este directorio contiene la configuración y utilidades para testing.

## Archivos

- `setup.ts`: Configuración global de tests (mocks, cleanup, etc.)
- `utils.tsx`: Helpers de renderizado con providers
- `__mocks__/`: Mocks personalizados

## Uso

Los tests se escriben junto a los archivos que prueban usando el patrón `*.test.tsx` o `*.spec.tsx`.

Ejemplo:
- `LoadingSpinner.tsx` → `LoadingSpinner.test.tsx`
